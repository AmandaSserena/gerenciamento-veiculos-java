# Gerenciamento de Veículos - em Java

Projeto simples para praticar **classes abstratas**, **herança** e **encapsulamento** em Java.

## Como executar
Requer **Java 17+** (funciona desde Java 8).

```bash
cd src
javac *.java
java Main
```

Saída esperada:
```
Carro: Toyota Corolla (2022) - 4 portas
Moto: Honda CB500F (2021) - 471cc
Após alterações:
Carro: Toyota Corolla (2023) - 4 portas
Moto: Honda CB500F (2021) - 500cc
```
